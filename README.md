# Car Sales Dashboard Analysis

## Objective
Analyze car sales data and build an interactive Power BI dashboard to monitor performance, identify trends, and support business decision-making.

## Tools Used
- Microsoft Excel
-	Power BI
-	Power Query
-	DAX

## Key KPIs
-	Total Sales: ₹5.54 Billion 
-	Total Orders: 1,000 
-	Total Units Sold: 5,517 
-	Average Sales per Order: ₹5.54 Million 

## Key Insights
-	East region generated the highest sales (₹1.50 Billion). 
-	Sedan category contributed the highest revenue (₹2.24 Billion). 
-	Sedan D was the top-selling product (₹1.18 Billion). 
-	Rahul was the top-performing salesperson (₹1.02 Billion sales). 
-	November recorded the highest monthly sales (₹518 Million). 

## Dashboard Features
-	KPI Cards
-	Region-wise Sales Analysis
-	Category-wise Sales Distribution
-	Monthly Sales Trend
-	Product Performance Analysis
-	Salesperson Performance Tracking
-	Interactive Slicers and Filters
